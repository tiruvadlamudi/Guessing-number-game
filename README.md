Number Guessing Game (Python)

A fun **command-line Number Guessing Game** built using Python.  
The program randomly selects a number between **1 and 100**, and the player must guess it within a limited number of attempts based on the chosen difficulty level.

 Project Description
This game allows users to:
- Choose a difficulty level (Easy or Hard)
- Guess a randomly generated number
- Receive hints whether the guess is too high or too low
- Win or lose based on available attempts

This project is ideal for **Python beginners and freshers** to practice logic building, loops, functions, and conditional statements.

 Technologies Used
- Python 3
- `random` module

 Features
- Random number generation
- Two difficulty levels:
  - Easy → 10 attempts
  - Hard → 5 attempts
- Feedback on each guess
- Game-over and win conditions

 How to Run the Project
1. Install Python 3 on your system
2. Clone the repository or download the file
3. Run the program using:

```bash
python number_guessing_game.py
